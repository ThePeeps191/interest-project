Frontend web development deals with the visual aspects of a websites using html, css, and javascript. For example, without frontend developers, there wouldn't be pictures, things that you could interact with, and the websites itself.

Skills you need to become a front end web developer are: Coding languages: HTML, CSS, JS Frameworks: Bootstrap, AngularJS, ReactJS, Ember Other: Git, Github

Anyone can become a frontend web developer and be hired in many companies. Most people will go too universities focusing on tech such as, Waterloo, MIT, Toronto, British Columbia, or any of the universities of california.
